Orekit 航天动力学库调用方法

Orekit 是一个基于 Java 语言开发的航天动力学库，采用对商业友好的 Apache 开源许可协议，第一个公开发行版本始于 2003 年。十多年来，Orekit 一直专注于航天动力学底层算法的实现，包含丰富的航天动力学元素：轨道、时间、参考架、姿态和事件等，以及大量处理这些元素的算法：元素转换、航天器状态预报、卫星姿态指向和事件响应等。
